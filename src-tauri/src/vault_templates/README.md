# JARVIS-OS

Sir's second brain: a local, git-versioned, plain-markdown vault organized
into three domains (work / business / personal). The JARVIS desktop app is
the single interface — it reads this vault for context on every conversation
and writes back (decisions, notes, context updates) only with confirmation,
auto-committing every change.

Start at `CLAUDE.md` (the routing tree). Live-system registry: `connections.md`.
