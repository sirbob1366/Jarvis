# /level-up

1. Read the latest audit (or run /audit if stale).
2. Pick the single weakest area across the three domains.
3. Interview sir for 3–5 targeted questions ONLY about that area.
4. Restructure that file: clear headings, current facts, dated notes moved
   to the bottom, contradictions resolved against decisions/log.md.
5. Confirm the rewrite (update_context), log a decision entry, and speak a
   one-line summary of what leveled up.
